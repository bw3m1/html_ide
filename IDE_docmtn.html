Getting Started

Welcome to the HTML IDE, your lightweight but powerful web development environment.
This editor gives you professional coding features with instant previews, all in your browser.

Built on Monaco Editor (the same tech that powers VS Code), you get:

  • Real-time HTML/CSS/JS editing with tab support
  • Instant visual feedback
  • All the essential tools without the bloat
  • Full VS Code keyboard shortcuts

File Operations

New File

Start fresh with a clean HTML template including basic structure and CSS reset.

How to: File → New or Ctrl+N (creates in current tab)
New Tab: Click the + button in tab bar or Ctrl+T

Open File

Work with existing files in these formats:

  • HTML (.html)
  • CSS (.css)
  • JavaScript (.js)
  • Plain Text (.txt)

Note: Opening a file will create a new tab if there are unsaved changes in current tab

How to: File → Open or Ctrl+O

Save Options

Save your work without hassle:

  • Save: Quick save to current file (Ctrl+S)
    Note: Silently saves and overwrites existing files. If nothing happens when you click it, that means it worked!
  • Save As: Choose format and location (Ctrl+Shift+S)
  • Export ZIP: Package your entire project

All operations affect the currently active tab

All options under File → Save menu

Editing Features

Smart Editor Basics

The editor understands your code and helps with:

  • Automatic indentation
  • Bracket matching
  • Syntax highlighting
  • Multiple cursors (Alt+Click)
  • Line operations (copy, move, delete)

VS Code Power Features

IntelliSense

Smart code completions that suggest HTML tags, CSS properties, and JS functions as you type.

Press Ctrl+Space to trigger manually.

Error Checking

Red squiggles under problems with hover explanations.

Works for HTML, CSS, and JavaScript syntax errors.

Code Folding

Collapse sections by clicking gutter arrows (left of line numbers).

Use Ctrl+Shift+[ and Ctrl+Shift+] to fold/unfold.

Command Palette

Press F1 to open the command palette - search for any VS Code action.

Special prefixes for quick access:

  • > - Run commands (default)
  • : - Go to line number (Ctrl+G shortcut)
  • @ - Jump to symbols (functions, classes, etc.)
  • # - Search across files

Code Navigation

Move through code efficiently with these essential shortcuts:

Ctrl+G Go to Line Ctrl+F Find Ctrl+H Replace Ctrl+D Select Next Match Ctrl+Shift+K Delete Line Shift+Alt+↓ Copy Line Down Shift+Alt+↑ Copy Line Up Ctrl+/ Toggle Comment

Tab System

Working with Multiple Files

The tab system allows you to work with multiple files simultaneously:

  • New Tab: Open a fresh editor tab (Ctrl+T)
  • Switch Tabs: Click any tab or use Ctrl+Tab to cycle forward and Ctrl+Shift+Tab to cycle backward
  • Close Tab: Click the × on the tab or use Ctrl+W
  • Tab Persistence: Open tabs are automatically saved and restored when you return
  • File Operations: All file operations (save, open, etc.) work with the currently active tab

Tab Features

Each tab maintains its own:

  • File content and unsaved changes
  • File name and type
  • Cursor position and scroll location
  • Language mode (HTML/CSS/JS/etc.)

Tip: You can have different layout modes (horizontal/vertical split) for different tabs

Preview Options

Live Preview

See changes as you type with automatic updates (500ms delay to prevent flashing).

The preview automatically refreshes when you:

  • Save the file (Ctrl+S)
  • Stop typing for half a second
  • Click the refresh button
  • Switch tabs

Layout Modes

Choose what works for your screen:

  • Horizontal Split: Editor on left, preview on right
  • Vertical Split: Editor above, preview below
  • Editor Only: Maximize code space
  • Preview Only: Full output view

Note: Layout settings are remembered per tab

How to: View → Layout or use the command palette (F1)

Run Options

Test your code different ways:

  • Current Tab: Update built-in preview (Ctrl+Enter)
  • New Tab: Open in browser tab (Ctrl+Shift+Enter)
  • Refresh: Manually reload preview

How to: Run menu or keyboard shortcuts

File Explorer

Project Management

Organize files without leaving the editor:

  • Add Files/Folders: Use the + buttons in the explorer toolbar
  • Delete Items: Click the × that appears when you hover
  • Toggle Visibility: File → Toggle Explorer or Ctrl+B
  • Drag and Drop: Reorganize files by dragging
  • Open in New Tab: Ctrl+Click files to open in new tab

Appearance

Themes

Switch between light and dark modes to match your environment.

How to: Settings → Theme or command palette (F1 then type "theme")

Editor Customization

Make the editor comfortable for you:

  • Fonts:
    • Monospace (default)
    • Arial (clean sans-serif)
    • Courier (classic terminal style)
  • Sizes:
    • Small (12px) - more code on screen
    • Medium (16px) - balanced size
    • Large (20px) - easier reading
  • Zoom: Adjust with Ctrl+=/Ctrl+-
  • Line Numbers: Toggle on/off via View menu
  • Tab Bar: Always visible at top of editor

Built-in Examples

Quick-start templates for common web patterns:

  • Basic HTML: Minimal page skeleton with proper doctype and meta tags
  • Form Example: Complete styled form with all input types
  • CSS Grid: Modern layout demo with responsive grid
  • Table Example: Proper data table with zebra striping
  • Responsive Design: Mobile-first starter with media queries

How to: Help → Examples

Tip: Examples open in new tabs so you don't lose your current work
