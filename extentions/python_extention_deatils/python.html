Python Extension
Run Python directly in your browser

Python Runtime Environment

In Development

This extension enables Python 3.11 execution in the browser using Pyodide, a WebAssembly port of Python. Execute Python code, import libraries, and visualize results without server-side processing.

Features

  • Full Python 3.11 Runtime - Standard library support with no server required
  • Scientific Computing - Includes NumPy, Pandas, Matplotlib, and SciPy
  • Interactive REPL - Execute code line by line in a console interface
  • Code Intelligence - Autocompletion, syntax highlighting, and linting
  • Package Management - Install packages with MicroPIP
